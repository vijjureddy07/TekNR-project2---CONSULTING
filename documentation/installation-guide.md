# Installation Guide

This guide will walk you through the steps to get your new template up and running.

## Prerequisites

Before you begin, you will need a code editor (we recommend [Visual Studio Code](https://code.visualstudio.com/)) and a local web server.

## Installation

1.  **Download and Unzip:** Download the template files from the marketplace and unzip them to a new directory on your computer.

2.  **Open in VS Code:** Open the unzipped directory in Visual Studio Code.

    ```
    code .
    ```

3.  **Run a Local Server:** To view the template, you need to run a local web server. We recommend the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension for VS Code.

    *   Install the Live Server extension from the VS Code marketplace.
    *   Right-click on any `.html` file in the `pages` directory and select "Open with Live Server".

4.  **Start Customizing:** Your template should now be running in your browser. You can start customizing the content by editing the HTML files in the `pages` directory.

## Next Steps

*   **Customization:** See the [Customization Guide](customization-guide.md) for instructions on how to customize the colors, fonts, and other design elements.
*   **Page Structure:** The [Page Structure](page-structure.md) guide provides an overview of all the pages included in the template.
*   **Deployment:** When you are ready to deploy your site, you will need to upload the files to a web hosting provider.
