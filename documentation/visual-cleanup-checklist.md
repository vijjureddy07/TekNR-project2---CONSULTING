# Visual Cleanup Checklist

## Global Review
- Verify the header and footer stay consistent on every public page.
- Check section spacing, container width, and vertical rhythm across desktop, tablet, and mobile.
- Confirm light mode and dark mode both keep strong text contrast.
- Check RTL toggle, theme toggle, and mobile navigation behavior on every page.
- Review button, card, image, and link hover states for consistency.
- Make sure no section feels unfinished, too empty, or overloaded.
- Verify no local CSS, JS, image, or page links are broken.

## Home
- Check the hero for clear positioning, strong visual hierarchy, and well-spaced CTAs.
- Make sure the page previews the rest of the site clearly through services, proof, and CTA sections.
- Review image sizing so the hero feels prominent without overpowering the text.

## About
- Confirm founder story, company positioning, values, and credibility are easy to scan.
- Make sure supporting imagery feels relevant and not decorative filler.
- Review section spacing so the page feels calmer than the homepage.

## Services
- Check the service overview for even card heights, aligned copy, and consistent CTA placement.
- Make sure naming and structure match the small-business consulting niche.
- Verify service sections do not leave unnecessary empty rows.

## Case Studies
- Confirm the proof story is clear through challenge, strategy, and result blocks.
- Review stats, quotes, and imagery for clarity and balance.
- Make sure CTAs route toward contact or services appropriately.

## Industries
- Check that each industry block feels distinct but still part of the same system.
- Review icon, image, or card treatment for consistency and spacing.
- Make sure the page does not repeat the services layout too closely.

## Team
- Confirm profile hierarchy, role clarity, and image consistency.
- Check bio lengths, social/contact actions, and responsive stacking.
- Make sure spacing does not create awkward empty gaps.

## Contact
- Confirm the contact form, direct contact details, and CTA order support conversion.
- Check validation states, field labels, and button alignment.
- Review map or booking placeholders if present.

## Final Sign-Off
- Verify one H1 per page and a clean heading hierarchy.
- Confirm each page has a title and meta description.
- Check mobile layout below 640px, tablet layout between 640px and 1024px, and desktop beyond 1024px.
- Review dark mode for low-contrast text or overly dark surfaces.
- Confirm all remaining design polish can be completed without restructuring the page layout.
